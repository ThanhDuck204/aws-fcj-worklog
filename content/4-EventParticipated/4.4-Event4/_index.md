### Speaker Lineup

#### 1. Steve Tran - AgenticOps for Your Cloud

**Title**: Founder of CloudThinker.io

**Journey from Zero to Founder**
Steve Tran shared his inspiring journey from zero to founding CloudThinker.io: how he started his cloud journey with no prior experience, steps to master cloud technology, building a successful cloud consulting company, and key lessons from his entrepreneurial journey.

**Microservices Architecture**
He covered microservices concepts in depth: core principles, when to use microservices, common anti-patterns to avoid, and industry best practices.

**Single Agent vs Multi-Agent Systems**
- Single Agent: simple tasks, limited scope, straightforward decision making, single domain expertise
- Multi-Agent: complex multi-domain problems, parallel processing, specialized agents, scalability and fault tolerance needs
- Key considerations: system complexity, scalability, development and operational costs, long-term maintenance, performance requirements

---

#### 2, 3, 4. Voice AI Team - Building Voice Agents at Scale

**Speakers**: Nghi Danh, Kiet Tran, Trung Vu (Founder & CEO of Revve AI)

The team provided a comprehensive view of the voice AI field, from market opportunities to practical product building.

**Voice Agent Market Opportunity**
Growing demand for voice interfaces, applications across industries, market trends, and business opportunities in voice AI.

**Workshop Content**
Hands-on experience with voice AI tools, building voice agents from scratch, integrating with existing systems, and best practices for voice interface design.

**Live Demo: Voice Agent**
Real-time voice interaction, natural language understanding, context-aware responses, multi-language support, and backend system integration. This was particularly impressive as it showed how far voice AI technology has come.

**Limitations and Solutions**
The team also addressed common Voice Agent limitations: difficulty with different accents and dialects, reduced performance in noisy environments, limited long-term context awareness, challenges in emotion detection and response, and handling multiple languages in one conversation.

Proposed solutions: diverse training datasets, advanced audio processing, better context management mechanisms, emotion analysis integration, and language detection and switching.

#### 9. Anh Toan & Anh Nghi - Building Secure Private MCP for Amazon Q

**What is MCP (Model Context Protocol)?**
A protocol connecting AI models with external data sources, standardizing context provision for AI models, enabling secure and controlled data access. Components: MCP Server, MCP Client, Context Provider, and Security Layer.

**Why MCP Matters**
Ensures data security and privacy, provides controlled access to sensitive information, enables consistent context across different AI tools, and supports compliance requirements.

**VPC Connection Solution**
Challenge: how to securely connect Amazon Q to private data sources? Solution: deploy MCP Server within VPC, use VPC Endpoint for secure connection. Benefits: data never leaves the private network, all communication encrypted, granular IAM permissions, full audit logging.

**Video Demo**
Setting up VPC infrastructure, deploying MCP server, configuring security settings, and connecting data sources. Amazon Q successfully queried private data through MCP, retrieving real-time context securely.

### Key Takeaways

**1. AI Agent Architecture**
Start with single agent for simple problems, scale to multi-agent for complex tasks. Consider cost and complexity from the start.

**2. Voice AI Technology**
Technology is advancing rapidly, market opportunities are growing. Understand limitations and plan for noise handling and voice diversity.

**3. AWS DevOps Automation**
AWS DevOps Agent offers native AWS integration advantages, AI-powered intelligent automation, and cost efficiency compared to traditional tools.

**4. Amazon Q**
Applications in HR operations automation, knowledge management, code assistance, and data analytics.

**5. Enterprise AI Security**
MCP architecture is crucial for enterprise AI adoption, with security by design, VPC-based isolation, and comprehensive audit capabilities.
